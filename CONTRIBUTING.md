# Contributing

Thank you for your interest in contributing to Unit Conversion Data.

We welcome contributions that improve the quality of this project.

## Ways to contribute

- Add new unit categories
- Improve existing conversion datasets
- Fix incorrect conversion factors
- Improve documentation
- Report issues
- Suggest new features

## Guidelines

- Keep data accurate.
- Follow the existing JSON format.
- Write clear commit messages.
- Test your changes before submitting.

Thank you for helping make this project better.
