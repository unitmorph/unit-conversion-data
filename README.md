# Unit Conversion Data

An open-source collection of unit conversion datasets, reference tables, and documentation for developers, students, engineers, and educators.

This project aims to provide reliable conversion data in a simple, structured format that can be used in web applications, mobile apps, APIs, calculators, and educational projects.

## Features

- Open-source conversion reference
- Structured JSON datasets
- Metric and Imperial units
- Easy to integrate into applications
- Developer-friendly format
- Community contributions welcome

## Categories

## Example

```json
{
  "category": "Length",
  "base_unit": "meter",
  "units": {
    "kilometer": 1000,
    "centimeter": 0.01,
    "millimeter": 0.001,
    "inch": 0.0254,
    "foot": 0.3048
  }
}
```
